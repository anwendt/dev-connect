## Approved Decisions Before Workflow Implementation

The following decisions are approved for the initial implementation:

1. CI shall target Go 1.26.x.
2. Linting shall use `golangci-lint` with `.golangci.yml`.
3. Vulnerability scanning shall use `govulncheck` and Trivy.
4. License scanning shall use `go-licenses`.
5. SBOMs shall use SPDX 2.3 generated by Syft as `sbom.spdx.json`.
6. Signing shall use Cosign with keyless Sigstore signing for public releases and optional enterprise key-based signing.
7. GHCR organization shall be `ghcr.io/anwendt/dev-connect`.
8. Release binaries shall be built before container images, and container images shall consume those binaries.
9. GitHub Actions permissions shall follow least privilege.
10. Temporary CI artifacts shall be retained for 30 days; release artifacts shall be retained permanently.
11. Automated dependency updates shall use Renovate Bot for Go modules, GitHub Actions, and Docker base images.
12. The release pipeline shall target SLSA Level 3 compatibility where feasible, using GitHub OIDC, reproducible builds, signed artifacts, Cosign, and provenance generation.
